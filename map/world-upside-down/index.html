World Upside Down

This example shows how to create a rotated world map with d3-geomap.

See the complete JavaScript code needed to render the map below.

var map = d3.geomap()
    .geofile('/d3-geomap/topojson/world/countries.json')
    .rotate([0, 0, 180])
    .draw(d3.select('#map'));
